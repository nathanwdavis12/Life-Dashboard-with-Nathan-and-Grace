# Life with Nathan and Grace

Baby gorgeous,

This website is meant to be a **central hub for anything we want** as we build our life together. The idea came to me thinking about graduation. Specifically, I thought of it as I started to build my own finance spreadsheet to help keep track of my expenses with my income. But then, I thought, "this is something that I could share with Grace because she would find it useful, too," and then I ended up creating this website so that we can not just organize our finances.....but EVERYTHING. Literally anything we want.

## The Idea

Instead of trying to remember everything or scatter plans across different places, this site gives us **one place where everything important can live**.

If we ever want a new section, we add it.

If something stops being useful, we change it.

The goal is simple:
**a shared system that grows with us.**
