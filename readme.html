README

Baby gorgeous, This website is meant to be a central hub for anything we want as we build our life together. The idea came to me thinking about graduation. Specifically, I thought of it as I started to build my own finance spreadsheet to help keep track of my expenses with my income. But then, I thought, "this is something that I could share with Grace because she would find it useful, too," and then I ended up creating this website so that we can not just organize our finances.....but EVERYTHING. Literally anything we want.

The structure of the site is intentionally flexible. Sections can be added, changed, or removed anytime depending on what we want to track or plan.

If there is something new we want to organize.....it can simply become a new section.

The goal is for this site to be a shared system that will grow with us.
